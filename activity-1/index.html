Rust Programming Language

What is Rust?

Rust is a multi-paradigm, general purpose programming language. Rust has a strong emphasis on:

  • Performance
  • Concurrency
  • Type safety
  • Memory safety

Rust is a statically-typed language, meaning type safety is checked during compilation instead of
while the program is running.

Rust is a systems programming language, meaning it can be used to write system software such as
systems and compilers.
